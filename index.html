Font Stacks

The font used on this page is Droid Serif

This site displays the common font stacks. Use the navigation above to see the example pages.

To actually see which font is being used on a page, select some text, right-click on the selection and choose 'inspect element' to open the dev tools panel, then look at the CSS font-family value. Often the used text is highlighted in some way e.g. underlined (FireFox). Also, you will be able to see the complete font stack.

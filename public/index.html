This site displays the common font stacks. Use the navigation above to see how the page's content looks using the chosen font stack.

To actually see which font is being used, select some text, right-click on the selection and choose 'inspect element' to open the dev tools panel, then look at the CSS font-family value. Often the font used is highlighted in some way e.g. underlined (FireFox). Also, you will be able to see the complete font stack.
